> **Organize. Browse. Preview. Your VALORANT Collection**

A modern, feature-rich desktop application for managing your VALORANT inventory. Track your skins, buddies, player cards, and sprays all in one place with beautiful visualizations and real-time statistics.

---

## ✨ Features

### 📊 Comprehensive Dashboard
- **Total Items**: 2,847+ items tracked
- **Real-time Statistics**: Monitor your collection growth
- **Visual Analytics**: Beautiful charts and graphs
- **Quick Overview**: All metrics at a glance

### 🎯 Advanced Inventory Management
- **Complete Catalog**: All skins, buddies, cards & more
- **Smart Filters**: Find exactly what you need
- **Favorites System**: Mark & access your favorites
- **Bulk Operations**: Manage multiple items at once

### 🎨 Skin Preview
- **3D Preview**: Inspect skins in high quality
- **Detailed View**: Zoom, rotate, and inspect
- **Comparison Mode**: Compare different skins
- **Real-time Rendering**: Smooth 60fps experience

### 📈 Statistics & Analytics
- **Collection Growth**: Track your progress over time
- **Value Tracking**: Monitor market values
- **Rarity Distribution**: Visual breakdown of collection
- **Performance Metrics**: K/D, win rate, headshot %

### 💰 Price Tracker
- **Market Monitoring**: Real-time price updates
- **Investment Analysis**: Track collection value
- **Price History**: Historical data visualization
- **Sale Alerts**: Never miss a good deal

## 🚀 Quick Start

### Prerequisites

| Requirement | Version |
|-------------|---------|
| Node.js | 16.0+ |
| npm | 7.0+ |
| Git | Latest |
| Electron | 27.0+ |

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/valorant-inventory-manager.git

# Navigate to project directory
cd valorant-inventory-manager

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
