# Valorant Inventory Manager (VIM)

![Valorant Inventory Manager](screenshots/main_mockup.png)

**Organize. Browse. Preview. Your VALORANT Collection.**

A powerful, beautiful and open-source inventory manager for your Valorant skins.

## ✨ Features

- **Inventory Grid** — Clean and responsive skin gallery
- **Skin Preview** — High-quality 3D-like skin viewer
- **Statistics Dashboard** — Detailed analytics of your collection
- **Advanced Filters** — Rarity, weapon, collection, price and more
- **Favorites & Price Tracker** — Keep track of your most valuable items
